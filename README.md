# houseLang

## Синтаксис
### Команды
`var` - установить переменную\
`print` - вывод текста на экран\
`exit` - выход из программы
### Данные
`"hello"` - строка\
`10` - число
### Символы
`_` - Пробел ( )\
`&_` - Нижнее подчёркивание (\_)

## Примеры
### Hello, world
```
var text = "Hello,\_World!"
print text
```
